Duck Typing

"If it walks like a duck and quacks like a duck, it must be a duck." Python doesn't check an object's type; it checks whether the object has the methods or attributes it needs. The class hierarchy is irrelevant — behavior is everything.

Static vs Dynamic Typing

                

                    In statically typed languages (Java, C++, Rust), types are
                    checked at compile time — errors are caught early but the code is more verbose.
                    In dynamically typed languages (Python, JavaScript), types are
                    checked at runtime — flexible and fast to write, but type errors only surface
                    when the offending line executes.

Strong vs Weak Typing

                

                    A strongly typed language refuses to silently mix
                    incompatible types — you must convert explicitly. A weakly
                    typed language is happy to coerce values behind your back, sometimes with
                    surprising results. Note this is independent of static vs dynamic:
                    Python is dynamic + strong, JavaScript is dynamic + weak.
