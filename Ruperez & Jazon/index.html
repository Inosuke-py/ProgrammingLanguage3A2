⚖️ Static vs Dynamic Typing Type Systems

          

            In statically typed languages (Java, C++, Rust), types are
            checked at compile time — errors are caught early but the code is more verbose.
            In dynamically typed languages (Python, JavaScript), types are
            checked at runtime — flexible and fast to write, but type errors only surface
            when the offending line executes.

💪 Strong vs Weak Typing Type Systems

          

            A separate axis from static/dynamic. Strong typing means the
            language refuses to silently mix types. Weak typing means
            the language will coerce values to make the operation work, sometimes with
            surprising results.

🧬 Inheritance Object Model

          

            A class can derive from one or more parents, reusing and extending their
            behavior. Python supports single, multi-level, and multiple inheritance,
            plus abstract base classes that force subclasses to implement specific methods.

📦 Encapsulation Object Model

          

            Bundling data and the methods that operate on it inside a class, while hiding
            internal details. Python relies on naming conventions
            (_protected, __private) and properties rather than
            hard access modifiers.

⚙️ Generator Function Simulator Function Mechanics

          

            A generator is a function with one or more yield
            statements. Calling it doesn't run the body — it returns a generator object.
