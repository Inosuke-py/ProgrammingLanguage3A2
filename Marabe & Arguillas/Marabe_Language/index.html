Duck Typing

Behavior over inheritance

Python checks whether an object supports the behavior a function expects. If it quacks like a duck, it can be used like one, even without sharing a class.

Static vs Dynamic Typing

            

            When checking happens
          

          

            Static typing catches problems during compilation or analysis. Dynamic typing defers type checks until execution, making code more flexible but shifting error discovery to runtime.

Strong vs Weak Typing

            

            Type coercion behavior
          

          

            Strong typing rejects silent type mixing. Weak typing often coerces values automatically, which can make code shorter but less predictable.
