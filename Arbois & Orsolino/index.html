Lexical Analysis is the first phase of the compiler. It reads raw source code character by character and groups characters into meaningful units called tokens.

How the Lexer scans:
def greet(x):
KEYWORD
def IDENTIFIER
greet DELIMITER
( IDENTIFIER
x DELIMITER
) DELIMITER
:
KEYWORD def if for return class Reserved — cannot be used as names
IDENTIFIER greet total myVar Variable, function, or class names
NUMBER 42 3.14 0xFF 4j Integer, float, hex, complex
STRING "hi" 'ok' f"x={x}" Text enclosed in quotes
OPERATOR + - == != && := Arithmetic, comparison, logical
LEX ERROR $ @ ` ? Unrecognised character — stops lexer

⚠ Comments (# in Python, // and /* */ in C++) are recognised by the lexer but excluded from the token output — they carry no semantic meaning.
