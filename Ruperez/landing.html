Built for understanding, not just running.

Five things that make Pyrism different from a plain code playground.

🔍

Real step-by-step trace

Every Python run can be replayed as an indented call tree of calls, returns, prints, and exceptions — driven by sys.settrace, not faked.

🌐

Five live engines

Pyodide for Python. ruby.wasm for Ruby. JSCPP for C/C++. Your browser's own engine for JavaScript. Trace works across all of them.

📚

Curated examples

Each concept ships with multiple worked examples — beginner all the way to gotchas — so you can pick the depth you need.

💡

Plain-language explanations

Every example pairs runnable code with a short note explaining the why — the kind of thing a good professor says out loud.

📱

Phone-first responsive

Sidebar collapses to a drawer, editor heights resize, tap targets stay reachable. Study on the bus, not just at a desk.

🚀

Zero install

Static files. No backend, no account, no database. Drop the folder on any host — or run python -m http.server locally.

A real runtime for every language.

Pick a language, click Run, see actual output — no server, no install.

🐍

Python live

Pyodide — CPython on WebAssembly. Includes the standard library and powers the real Trace via sys.settrace.

💎

Ruby live

ruby.wasm — Ruby 3.3 compiled to WebAssembly. Lazy-loaded on first Run. Stdout captured via StringIO.

📜

JavaScript live

The browser's own engine. Always available, zero download. Captures console.log calls into the output panel.

🔧

C live (subset)

JSCPP — a JavaScript-based C interpreter. Handles stdio, recursion, basic data types. Loads on first C Run.

⚙️

C++ live (subset)

JSCPP — supports iostream, classes, basic templates, and STL containers. Modern features like concepts and coroutines fall back to a simulated output.

How it works

Three clicks from "huh?" to "ohhh".

1

Pick a concept

The sidebar groups topics by Type Systems, Object Model, and Function Mechanics. Pick one to load a panel with a worked example.

2

Choose a language

Each tab has a segmented switcher: 🐍 Python · 💎 Ruby · 🔧 C · ⚙️ C++ · 📜 JavaScript. Examples and editor mode swap instantly.

3

Run or Trace

Run shows the output. Trace indents one level per call so you can watch the stack grow and unwind step by step.
